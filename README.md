# research-report

研报下载助手

## Usage

```text
❯ research-report -h
Usage of research-report:
  -begin-time string
      开始时间 (default "2021-09-04")
  -download-path string
      下载路径 (default "/tmp/")
  -end-time string
      结束时间 (default "2021-10-04")
  -min-pages int
      研报最小页数 (default 20)
  -query-type string
      研报类型，0：个股研报，1：行业研报，2：宏观研究 (default "0")
```
