# research-report

研报下载助手

## Usage

```text
❯ research-report -h
Usage of research-report:
  -begin-time string
        开始时间 (default "2021-10-08")
  -download-path string
        下载路径 (default "/tmp")
  -end-time string
        结束时间 (default "2021-10-09")
  -min-pages int
        最小页数 (default 15)
  -query-type string
        研报类型，0：个股研报，1：行业研报，2：宏观研究 (default "0,1,2")
```
